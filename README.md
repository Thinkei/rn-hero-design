## Development

- Add your components and themes to `lib` folder
- Create your stories in `playground` folder
- Start the playground bundler
```
yarn dev
```
- Deploy the playground to expo
```
yarn playground:publish
```

## Documentation

- Start dev server
```
yarn doc:dev
```
